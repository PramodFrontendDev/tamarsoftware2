Senior Frontend Engineer

React / Next.js Noida / Remote Full-Time

What you will do:

  • Build highly responsive and interactive client-facing dashboards using React, Next.js, and TypeScript.
  • Collaborate with UI/UX designers to translate Figma mockups into premium, clean HTML and vanilla CSS.
  • Optimize web applications for maximum loading speed and Interaction to Next Paint (INP) scores.

What we look for:

  • 3+ years of experience with React, state management (Redux/Zustand), and modern CSS layouts.
  • Proficiency in RESTful and GraphQL API integrations.
  • A strong eye for visual layout detail, margins, fonts, and responsive grid alignments.

Full Stack Web Developer

Node.js / React Noida / Remote Full-Time

What you will do:

  • Develop robust backend services and REST APIs using Node.js, Express, and PostgreSQL/MongoDB.
  • Maintain database security rules, read-write indexes, and handle concurrent query optimizations.
  • Build admin interfaces and connect backend databases to React frontend dashboards.

What we look for:

  • 2+ years of professional backend development with Node.js and SQL/NoSQL databases.
  • Familiarity with JWT auth systems, API security rules, and server deployment.
  • Basic knowledge of Docker and containerization is a plus.

DevOps & Cloud Engineer

AWS / Kubernetes Noida / Remote Full-Time

What you will do:

  • Configure and maintain staging and production servers on AWS, Microsoft Azure, or GCP.
  • Set up automated CI/CD pipeline triggers using GitHub Actions or Jenkins.
  • Monitor server loads, configure database auto-backups, and optimize server bills.

What we look for:

  • Familiarity with Docker containers, Kubernetes orchestration, and Linux shell commands.
  • Experience setting up Nginx proxies, SSL certificates, and firewall policies.
  • Knowledge of Terraform or Ansible infrastructure-as-code is highly valued.
